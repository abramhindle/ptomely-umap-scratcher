Class: CMRecorder

CMRecorder

new CMRecorder()

Recorder plugin for Canvas Mouse.

Records original points based on events and converted points (original points can be recalculated in case matrix for context changes).

Methods

clear()

Clear all recorded points and events.

handler(event, pos)

Mandatory handler called internally by Canvas Mouse's getPos(). Should never be called manually.

originalStrokes()

Returns non-calculated points based on the original event objects. The array contains arrays representing strokes. Each stroke array contains point objects. Each stroke array is intended to be a continuous stroke.

Returns:
Type
Array

recalc(cm)

Recalculate original events into new points based on the current state of the provided CanvasMouse instance.
